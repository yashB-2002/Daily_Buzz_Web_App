# Daily_Buzz_Web_App

## This is a Social Media type of Web Application build using MERN Stack.
### Features of the Web App -
- Register/login functionality
- Home Section to show all the post from users
- Create Post link where user can create their post
- profile Section in which all the posts user added is shown
- User can like/dislike other user post
- User can delete their added post
- User can add the comment on other user post.

### Key things required to build this web app -
-  How to build backend APIs with Node.js & Express.
-  All the feature of react.
-  JWT based authentication.
-  Images upload using Cloudinary.
-  Managing data with mongodb.
-  Testing API using Thunderclient.
